# 极简动态家庭资产配置与收益率计算看板 (v3.0)

本项目是一个高度可定制、极简高级感的**单文件版**网页应用（HTML+CSS+JS 混编）。无任何外部后端依赖，专为家庭资产结构梳理、跨资产类型统计及收益率分析量身打造。

## 🌟 核心功能特性

1. **无限级 Portfolio 自定义**：
   - 允许用户自由创建、命名及移除投资组合舱位（如：“子女教育基金”、“长线养老持仓”、“全纳防御存款”）。
2. **多资产组合动态挂载**：
   - 每个 Portfolio 下支持不限数量的资产明细项增删。
   - 允许自由录入如“东方30”、“国债2401”等具体标的别名。
3. **标准五大资产大类归集**：
   - 支持将所有标的指派归入：`固定收益`、`权益`、`基金`、`现金`、`其他`。
   - 图表全自动按底层资产大类穿透合并，输出精细的环形资产占比。
4. **月/年双周期滚动投资收益率计算**：
   - 顶部集成了复合金融看板，自动比对你当前录入的“当前总资产”与历史存根中 **“刚好1个月前”** 及 **“刚好1年前”** 的数据快照。
   - 自动得出准确的月度收益率（1M Return）与年度收益率（1Y Return），并自动高亮呈现涨跌红绿（涨：绿，跌：红）。
5. **本地原生级持久化**：
   - 利用浏览器进程级别的 `localStorage` 自动异步存储，无任何隐私上传泄露风险。

## 🛠️ 快速上手与使用指南

### 1. 启动方式
1. 将上文提供的 `index.html` 源码完整复制。
2. 在本地电脑新建文本文档，粘贴代码后重命名为 `family_wealth.html`。
3. 双击此文件，即可在任意主流现代浏览器（Chrome、Edge、Safari）中本地零延迟流畅运行。

### 2. 初始化你的资产架构
- **第一步**：点击 **“＋ 新增 Portfolio”** 设定你的账户母分类。
- **第二步**：在相应的分类框内点击 **“＋ 添加资产项”**，填入你的持仓名称，选择资产大类，并输入最新的账面资产净值。
- **第三步**：系统会自动在顶部汇总“家庭资产总净值”，并在右侧实时渲染大类比例。

### 3. 如何精确激活“收益率看板”？
收益率的计算极度依赖于你提供的**历史基期对比存根**：
- **想要看到 1M（1个月）收益率**：请在下方的“历史资产快照存根”中，选择 **当前月份的向前数第 1 个月**（如当前是 2026-06，则选择 2026-05），并填入你那个月末的资产总额，点击“记录快照”。
- **想要看到 1Y（1年）收益率**：请同理在存根中录入 **刚好 1 年前所在月份**（如 2025-06）的总资产。
- 录入完成后，顶部的 `--` 会立刻激活并刷新为真实的百分比数据。

## 📂 技术栈选型
- **UI 样式**：微原子化原生 CSS 变量 + 全响应式网格布局（自动兼容 iPhone/iPad/PC 屏幕）。
- **图表数据隔离层**：Chart.js (v4.x 异步安全分发版)。
- **持久层逻辑**：基于状态机 `appState` 的一键本地序列化方案。
